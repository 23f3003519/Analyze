Python Data Processing Project

A demonstration of Python scripting, data conversion, and GitHub Actions CI/CD.

Project Overview

This project demonstrates a simple data processing workflow:

  1. An Excel file (`data.xlsx`) is converted to CSV format (`data.csv`).
  2. A Python script (`execute.py`) reads `data.csv`, performs aggregation, and outputs a JSON result.
  3. A GitHub Actions workflow (`.github/workflows/ci.yml`) automates linting (Ruff), execution of the Python script, and publishes the `result.json` to GitHub Pages.

All necessary files are provided below for download, with execute.py containing the required fixes and data.csv being the direct conversion of the provided data.xlsx.
